# projeto-react

Projeto desenvolvido em React com Vite, apresentando um cardápio de lanchonete raiz com 5 itens.

## Funcionalidades
- Exibição de 5 produtos da lanchonete
- Nome e preço de cada item
- Layout estilizado em um único arquivo `App.jsx`

## Tecnologias utilizadas
- React
- Vite
- JavaScript

## Como executar o projeto
1. Clone este repositório
2. Entre na pasta do projeto
3. Instale as dependências com `npm install`
4. Execute com `npm run dev`

## Observação
O projeto foi desenvolvido com todo o conteúdo concentrado em um único arquivo, conforme solicitado na atividade.
